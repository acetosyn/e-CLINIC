How this workspace works

  • Use the search bar for instant lookup by name, file number, or patient ID.
  • Click Fetch Records to load a paginated, Excel-style table from Supabase.
  • Click a table row or search suggestion to open the Patient File modal.
  • Inside the Patient File modal, add services and see a live visit summary on the right.
